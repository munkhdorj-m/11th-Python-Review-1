## Exercise 1

**Problem:**

Input a number and check if the last digit is even or odd.

**Example:**

    Input: 123
    Output: Odd
    
    Input: 246
    Output: Even
    
    Input: 0
    Output: Even

## Exercise 2

**Problem:**

Check if a number is positive or negative.

**Example:**
    
    Input: 10
